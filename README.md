
# Info
FrontEnd with Svelte

# Setup
* Install node package: npm i
* Run: npx vite dev

# Command
- start: npx vite  
- build: npx vite build  
- xem trước: npx vite preview  
- Kiểm tra package có phiên bản mới: npm outdated  
- Cập nhật package mới nhất: npm i <name_package>@latest
